Quick Setup

To quickly scaffold control structures, create a simple sitemap in the text box below as illustrated by the following example:

  • FolderName/PageName
  • PageName
  • Account/Login
  • My/Profile
  • Search/Users
  • Search/Books
  • Legal
  • Checkout

The front page and 404 page are created for you. Folders should be no further than one level deep.
